Best game!

  • Originality: Digital 2 Rendering of Analog Iteration 1
  • Tutorial:
    • 1. Run to collect all the jewels before the zombie does
    • 2. Collect speed booster to run faster, there are 3 total
    • 3. Random flying Bats can shoot either HP damager: to reduce HP or de-speeder: to reduce your upgrades by 1. Make sure to dodge those with your speed.
    • 4. Zombie will start running from beginning to end, then back to beginning. Make sure to collect more Diamond before he does
    • 5. Collecting apple will grant you HP boost
    • 6. Game will end once one character collect 50 treasures
    • 7. Zombie can only collect treasures, so boosts are all yours!
